🏨 **Hotel Booking Cancellation Prediction**

🚀 **Predicting booking cancellations using Machine Learning**

---

📌 **Overview**

This project focuses on analyzing hotel booking data to identify the key factors that lead to cancellations. A machine learning model is built to predict whether a booking will be canceled, helping hotels reduce revenue loss and improve planning.

---

🎯 **Objective**

Identify factors influencing booking cancellations
Build a predictive classification model
Provide actionable insights to reduce cancellations

---

📂 **Dataset Features**

* Booking_ID – Unique booking identifier
* no_of_adults / no_of_children – Guest count
* no_of_weekend_nights / no_of_week_nights – Stay duration
* type_of_meal_plan – Selected meal plan
* required_car_parking_space – Parking requirement
* room_type_reserved – Room category
* lead_time – Days between booking and arrival
* arrival_year / month / date – Booking timeline
* market_segment_type – Customer segment
* repeated_guest – Returning customer flag
* no_of_previous_cancellations – Past cancellations
* avg_price_per_room – Room pricing
* no_of_special_requests – Customer preferences
* booking_status – Target variable (Canceled / Not Canceled)

---

🛠️ **Approach**

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Building (Classification Models)
* Model Evaluation & Comparison

---

📊 **Key Insights**

* Higher lead time increases cancellation probability
* Customers with previous cancellations are more likely to cancel again
* Pricing and special requests influence customer commitment
* Repeated guests are less likely to cancel

---

🛠️ **Tech Stack**

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

📈 **Outcome**

* Built a predictive model for booking cancellations
* Identified key drivers impacting cancellations
* Provided insights to improve revenue management

🚀 **Business Impact**

Helps hotels to:

* Reduce last-minute cancellations
* Optimize pricing strategies
* Improve resource planning
* Enhance customer retention
